# ccodexr 1.2.0

`v1.2.0` 是当前 `ccodexr` 公开仓库下的首个完整开源整理版本。

## 重点更新

- 统一项目品牌为 `ccodexr`
- provider / 账号切换后，尽量续接原有聊天线程
- 本地成本统计支持今日请求明细、请求 / 缓存 / 回答 token 拆分与逐条成本展示
- 今日趋势柱状图切换为 `08:00` 之后的时段视图
- 状态栏 helper 自动刷新更稳，减少“编译了但菜单栏没变”的情况
- 开源仓库补齐来源说明、许可证和下载入口

当前仓库附带的是 `guided download` 形式的发布资产。

- 应用内会读取 `release-feed/stable.json`
- 当前版本会提示新版本并打开下载入口
- 还没有接入完整的自动替换 / 自动重启 updater 闭环
